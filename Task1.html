HTML tag examples

HTML tags are special words wrapped in angle brackets (< >) which used to define different types of content in a webpage.

The browsers use these to decide how to render or display the following content on the screen.
Example:
  • <p>
  • <h1>
  • < img>,etc
  1. Create bold or italic text.
  2. Change the text color
  3. Change the font family or font size
  4. Create hyperlinks
  5. Create a bulleted list...
  6. ....and much more!

In this section, we will only discuss the tags which are valid in HTML5.
There are many tags that are deprecated in HTML5 like < center > and are not used.
